简述 int thread_join (thread) 的行为：

  • 如果 thread 未执行完毕，当前线程转入 WAITING；
  • 如果 thread 已执行完毕，返回该线程 thread_exit 参数；

Wait for thread to finish if it has not already done so; then return the value passed to thread_exit by that thread.

Note that thread_join may be called only once for each thread.

对于每个线程来说，可能只能有一个线程对其调用 thread_join
