设备控制器本身就会直接向内存中写入数据，只不过会在传输完成的时候会使用中断通知 CPU 

执行 IO 操作时，是由谁来向内存中写入数据？设备控制器

 

没有 DMA 时，设备控制器会以什么样的频率中断 CPU？每写入一个 Byte 中断一次

 

有 DMA 时，设备控制器会和谁进行交互？DMA

 

有 DMA 时，CPU 直接交互的对象是？DMA 控制器

 

DMA 会在什么时候通知 CPU？批量读取指令执行完毕

 

CPU 会如何使用 DMA 批量读取数据？向 DMA 控制器中写入批量读取数据的指令

现在有了 DMA，CPU 就会直接向 DMA 控制器中写入批量读取数据的指令，随后 DMA 就会将指令传输给磁盘控制器。等这些指令执行完成，所有需要的数据都被读取到内存之后，DMA 控制器就会中断 CPU：指令执行完成
