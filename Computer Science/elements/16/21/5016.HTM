When the operating system starts up, it ﬁrst runs threads/init.c.

当 OS 启动时，它首先会运行 threads/init.c

If the command-line arguments to Pintos indicate that the user wants to start running user programs, the run_task function is called

如果命令行参数表明用户想要启动用户程序，那么 run_task 函数就会被调用

which has the main thread of Pintos (the one running the threads/init.c code) call process_wait(process_execute(task)). So, the ﬁrst user program is created by tha main thread of the OS.

run_task 会让 Pintos 的主线程（也就是初始线程）调用 process_wait(process_execute(task))
