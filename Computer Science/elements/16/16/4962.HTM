Once that is done, thread creation code puts the thread into the READY state by adding the thread to the ready list.

初始化完成之后，用于 thread_create 就会将线程添加到等待列表中

The ready list is the set of runnable threads that are waiting their turn to use a processor.

本质上来说，位于等待列表的线程是一系列准备好使用处理器的线程

In practice, as discussed in Chapter 7, the ready list is not in fact a “list”; the operating system typically uses a more sophisticated data structure to keep track of runnable threads, such as a priority queue

实践中，等待列表往往并不是真正的列表，而是更加复杂的数据结构
