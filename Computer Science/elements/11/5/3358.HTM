Figure 4.8: A multi-threaded process or operating system kernel has both per-thread state and shared state.

多线程进程以及操作系统内核进程拥有两种不同类型的状态：线程状态和共享状态

操作系统内核也能被认为是多线程进程？

The thread control block stores the per-thread state

线程控制块（thread control block，TCB）中保存着线程状态（而不是共享状态）

the current state of the thread’s computation (e.g., saved processor registers and a pointer to the stack) and metadata needed to manage the thread (e.g., the thread’s ID, scheduling priority, owner, and resource consumption).

线程状态由两部分组成：线程当前的计算状态（寄存器和栈指针）以及维护线程所需要的元数据（线程 ID、调度优先级、拥有者、资源消耗）

Shared state includes the program’s code, global static variables, and the heap.

共享状态包括程序代码，全局静态变量以及堆

也就是说，每个线程有自己的栈，但是同进程的线程共享堆
