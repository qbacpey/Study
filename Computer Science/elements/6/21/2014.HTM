Title: AT&T 语法

寄存器的前面必然有一个什么符号？%

 

立即数的前面必然有一个什么符号？$

 

指令的格式一般为？inst src, dest

 

如我们想要访问内存地址 base + index * scale + offset 的话，需要使用什么语法？offset(base, index, scale)

 

offset(base, index, scale) 中的 base 和 index 需要是什么？寄存器

 

offset(base, index, scale) 中的 offset 和 scale 需要是什么？整数

offset(base, index, scale) 中的 scale 取值有什么特点？只能是1、2、4、8

 offset(base, index, scale) 所有参数都需要给定吗？不是，所有参数都是可选的

 

指令 mov %ecx, -4(%esi, %ebx, 8) 表示什么意思？将寄存器 ecx 中的内容移动到地址 esi + ebx * 8 -4

lea 所操作的对象是地址本身而不是地址中所保存的内容

指令 lea 8(%ebx), %eax 表示什么意思？将地址 ebx + 8 放在 eax 中

如果我们想让 mov 移动指定大小的数据，可以在 mov 后面加上数据大小后缀

指令的数据大小后缀包括？b、w、l
