Title: Dynamic Linking

The dynamic linking process resolves references either at process initialization time and/or at execution time.

动态链接过程既可能发生在进程初始化时间，也可能发生在进程执行时间，也有可能在两个时间都发生

动态链接最重要的目的是解决一些引用问题

Some basic mechanisms need to be set up for a  particular linkage model to work, and there are ELF sections and header elements reserved for this purpose.  

The actual definition of the linkage model is determined by the operating system  and implementation.

Therefore, the contents of these sections are both operating system and processor specific.  (See the appendix at the end of Book III.)
