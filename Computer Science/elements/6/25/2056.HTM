If the system call has a pointer argument (e.g., a file name or buffer), the stub must check the address to verify it is a legal address within the user domain. If so, the stub converts it to a physical address so that the kernel can safely use it

如果系统调用有一个指针参数，内核桩程序首先要将验证地址是否合法（是否位于用户的地址空间内），如果是，那么桩程序就会将该指针的地址转变成物理地址

当系统调用参数是指针的时候，桩程序主要检查指针所指地址是否位于用户的地址空间内

如果指针参数通过桩程序的测试，那么桩程序就会将该指针的地址转变成物理地址
