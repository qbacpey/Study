Title: Introduction : 定义 : OS 的工作

那么，OS的角色（或者说其工作）具体是什么呢：

  • Illusionist（幻术师）— OS 对硬件进行抽象以简化对于应用程序设计者的要求：

    • 使编码操作无需关注硬件细节（内存的大小、核心数目）；

    • （经由 OS 的资源调度）能更加灵活的利用硬件资源；

    • 高级对象：文件、用户、消息；

  • Referee（裁判）— OS 将协调各进程对于硬件的使用情况：

    • 协调各应用程序对硬件资源的使用，既包括高效利用也包括使用许可；

    • 防止某应用程序的错误影响到另一应用程序；

  • Glue（胶水）— OS 以提供共用服务（common services）的方式简化应用程序之间的共享操作：

    • 对于不同计算机上所使用的不同 IO 设备，由于这些计算机上针对硬件和应用程序的共用服务都是相同的，因此同一应用程序在这些计算机上都能被使用；
