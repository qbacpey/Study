To create a thread, the user library allocates a user-level stack for the thread and then does a system call into the kernel.

在 One-to-One 型线程系统中，如果用户级线程库想要创建线程，那么首先会在用户进程空间中分配用户级栈，随后执行系统调用进入内核

The kernel allocates a TCB and interrupt stack, and arranges the state of the thread to start execution on the user-level stack at the beginning of the requested procedure.

在 One-to-One 型线程系统中，如果用户级线程库想要创建线程，内核的工作是在内核空间中分配 TCB 和中断栈，随后设置线程从桩程序处启动

The kernel needs to store a pointer to the TCB in the process control block; if the process exits, the kernel must terminate any other threads running in the process.

此外，内核还需要在用户进程的 PCB 中设置指向线程 TCB 的指针，为此当用户进程退出的时候，内核需要终结在进程上运行的所有线程
