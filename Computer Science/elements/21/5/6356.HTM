Increasing the number of virtual processors. When a program starts, it receives an activation to inform the program that it has been assigned a virtual processor: that activation runs the main thread and any other threads that might be created.

当程序启动时，它会接受到一个激活，通知该程序已经被给予了一个虚拟处理器，程序就能在该虚拟处理器上运行主线程或者其他线程了（增加虚拟处理的数目）。之前提到每一个物理处理器上都由一个内核线程在运行调度器，还是感觉这里所谓的虚拟处理器实际上就是之前提到的内核线程，毕竟之前提到的内核线程调度那块，就是由调度器来执行内核线程的，所以这里内核线程执行用户级调度器之后，用户级调度器在内核线程的环境下执行用户线程的代码，应该这样来理解这个东西
