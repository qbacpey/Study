Each thread struct represents either a kernel thread or a user process. In each of the 3 projects, you will have to add your own members to the thread struct. You may also need to change or delete the deﬁnitions of existing members.

每个线程结构体表示一个内核线程或一个用户进程（在往后的三个项目中，都需要对线程结构体添加某些内容或者删除某些内容）

Every thread struct occupies the beginning of its own 4KiB page of memory. The rest of the page is used for the thread’s stack, which grows downward from the end of the page. It looks like this:

每一个线程结构体占据着分配给这个线程的 4KiB 内存页的起始位置，内存页中的余下部分会被用于作为线程的栈，它会从栈的顶部往下增长
