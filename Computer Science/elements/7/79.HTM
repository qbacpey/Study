If multiple interrupts arrive while interrupts are disabled, the hardware delivers them in turn when interrupts are re-enabled.

如果在禁用被中断的期间到达了多个中断，那么硬件会按照它们到达的顺序将它们分发到处理器

However, since the hardware has limited buffering for pending interrupts, some interrupts may be lost if interrupts are disabled for too long a period of time.

由于硬件中断缓存的空间是有限的，因此如果中断被禁用的时间太长，或者同时有太多中断到达，那么可能有一些中断会被丢失

Generally, the hardware will buffer one interrupt of each type;

硬件会保存不同类型的中断，但是硬件只会保存一个同类型中断 

the interrupt handler is responsible for checking the device hardware to see if multiple pending I/O events need to be processed

中断处理程序负责检查硬件设备，以确认是否有多个I/O中断等待处理
