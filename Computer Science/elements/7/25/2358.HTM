Title: x86 Assembly Guide : Declaring Static Data Regions

Data declarations should be preceded by the .data directive. Following this directive, the directives DB, DW, and DD can be used to declare one, two, and four byte data locations, respectively

如果我们想要在汇编语言文件中声明数据，需要在其之前加上什么？.data

如果在 .data 指令中出现 DB 代表指令声明的数据是以 Byte 为单位的

如果在 .data 指令中出现 DW 代表指令声明的数据是以 Word 为单位的

如果在 .data 指令中出现 DD 代表指令声明的数据是以 Double Word 为单位的
