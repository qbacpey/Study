The #include <...> notation causes the compiler to search for the ﬁle in the include paths speciﬁed with -I at compile time and system paths.

#include <...> 会使编译器在 -I 所声明的路径以及系统路径中搜寻文件

-I 所声明的搜寻路径也被称为 include 路径

In contrast, the #include "..." notation causes the compiler to search for the ﬁle in the current directory ﬁrst, before searching in the include paths and system paths.

#include "..." 会使编译器先在当前目录中搜寻文件，尔后才在 include 路径和系统路径中搜寻文件
