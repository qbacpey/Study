A program header table, if present, tells the system how to create a process image.

程序头表（program header table）的主要作用就是告诉系统如何构建进程映像

Files used to build a process image (execute a program) must have a program header table; relocatable files do not need one

所谓构建进程映像实际上可以被认为是执行程序一个必不可少的环节

用于构建进程映像的文件必须拥有程序头表，重定位文件不需要程序头表
