GDB 原生也支持一些特别的操作符：

如果我们想要打印一片连续内存区域中的值，可以使用什么操作符？<element>@<length>

<element>@<length> 中的 <element> 是整个数组的什么？第一个元素

如果我们想要打印一个地址为 array 元素个数为 length 的数组，可以使用什么指令？p/x *array@length

  • <element> @ <length>：这种语法一般用以指代程序中动态分配空间的数组（malloc）值与内存-1，<element>指的是解引用数组地址后得到的值（*array）,<length>指的是数组的长度（元素个数），效果如下所示：

     (gdb) p/x *array@len
     $1 = {0x1234, 0x5678}

  • [file|function]::variable：指代特定作用域中的静态变量；

  • {type} addr：指代地址为 addr 的地方有一个类型为 type 的变量；

如果我们想要指代特定作用域中的静态变量，可以使用什么语法？[file|function]::variable

如果我们想在指定变量的时候，同时指定其类型，可以使用什么语法？{type} addr
