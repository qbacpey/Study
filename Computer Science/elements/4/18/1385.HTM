Optionally save an error code. Certain types of exceptions, such as page faults, generate an error code to provide more information about the event; for these exceptions, the hardware pushes this code, making it the top item on the stack. For other types of events, the software interrupt handler pushes a dummy value onto the stack so that the stack format is identical in both cases.

保存错误代码是整个上下文切换过程的第五步

三个关键值被保存到中断栈之后，硬件会将错误代码保存到中断栈中

中断的类型不同，所保存的错误信息会有所差异

对于能够生成错误代码（error code）的中断来说（如page fault），硬件会直接把这些错误代码保存到栈中

对于那些不会生成错误代码的中断，软件中断处理程序会将一些冗余数据保存到栈中，以确保格式一致
