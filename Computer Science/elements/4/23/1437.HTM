Allocating a kernel stack per process makes it easier to switch to a new process inside an interrupt or system call handler.

For example, a timer interrupt handler might decide to give the processor to a different process. Likewise, a system call might need to wait for an I/O operation to complete; in the meantime, some other process should run.  

为每一个进程都分配一个中断栈有什么优点？

能让中断处理程序以及系统调用处理程序在其执行过程中更容易切换执行其他进程

举个例子？

计时中断处理程序可能在其执行过程中决定切换到其他进程的执行；系统调用处理程序等待I/O响应时，可以让处理器切换执行其他进程
