To carry out an input, output or memory-to-memory operation, the host processor initializes the DMA controller with a count of the number of words to transfer, and the memory address to use.

CPU 会使用将要传输的数据的数目，以及相关的内存地址来初始化 DMA 控制器

The CPU then commands the peripheral device to initiate a data transfer. The DMA controller then provides addresses and read/write control lines to the system memory.

CPU 随后会命令周边设备初始化数据传输，DMA 控制器则会把地址和读写控制线提供给系统内存

Each time a byte of data is ready to be transferred between the peripheral device and memory, the DMA controller increments its internal address register until the full block

每当一个 byte 的数据准备好在周边设备和内存之间进行传输了，那么 DMA 控制器就会递增内部的地址寄存器，直到数据块满为止（这里的数据块是内存数据块吗）

说到底，如果数据块满了 DMA 会做什么？重新向 CPU 申请内存吗也就是 DMA 使得 CPU 传输数据的单元从字节变成了 block
