Transition to WAITING. When a user-level thread blocks in the kernel waiting for I/O, the kernel similarly makes an upcall to notify the user-level scheduler that it needs to take action, e.g., to choose another thread to run while waiting for the I/O to complete.
当某个用户级线程因为 IO 事件或者别的什么事件（thread_join）被阻塞时，内核会激活用户层调度器，让它做出一定的行动（比如说让其他线程使用这个虚拟处理器？）
