One popular model is data parallel programming, also known as SIMD (single instruction multiple data) programming or bulk synchronous parallel programming. In this model, the programmer describes a computation to apply in parallel across an entire data set at the same time, operating on independent data elements.

数据并行编程（data parallel programming）也称为 SIMD（single instruction multiple data）编程
