Title: Calling Convention : Instructions

pushl src 指令的意义是什么？将 src 寄存器中的内容压入栈中

pushl src 和 popl dest 是否会修改栈指针？会

popl dest 指令的意义是什么？将栈顶的内容弹出到 dest 中

call addr 指令的意义是什么？保存当前指令地址并跳转到 addr

 

leave 通常和什么指令搭配使用？ret

leave 指令的意义是什么？将 EBP 和 ESP 中的内容恢复到先前的值

leave 指令会将 EBP 恢复成什么？Caller 栈帧的底部

leave 指令会将 ESP 恢复成什么？Caller 栈帧的顶部

 

leave 指令实际上相当于什么指令的执行效果？movl %ebp, %esp；popl %ebp

 

ret 指令的意义是什么？将返回地址弹出并跳转到该位置

ret 指令相当于什么指令的执行效果？popl %eip

如果想让 GCC 编译时确保 16-byte 的栈对齐，需要加上？-fno-ipa-stack-alignment
