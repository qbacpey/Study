A shell is a job control system; both Windows and UNIX have a shell.

shell 本质上是一个作业控制系统

Many tasks involve a sequence of steps to do something, each of which can be its own program. With a shell, you can write down the sequence of steps, as a sequence of programs to run to do each step. Thus, you can view it as a very early version of a scripting system

shell 允许我们将一些小的操作组装起来，让它帮我们完成一些大的工作
