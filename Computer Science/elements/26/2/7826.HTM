• 主机照常将数据报发送到局域网中：内网地址为 10.0.0.1 的主机将目标地址（128.119.40.186）和随机选定的端口号（5001）封装到数据报并发送到局域网中；

  • NAT 路由器对数据报进行处理：路由器接受到数据报以后，会将其原来的端口号（5001）和源地址（10.0.0.1）替换成新的端口号（3345）和路由器 WAN 侧 IP 地址（128.119.40.186）：

    1. “新” 端口号实际上是路由器从 NAT 表中选的一个未被使用过的端口号；
    2. NAT 表中的条目由五个部分组成：外网 IP 地址、“新” 端口号、WAN 侧 IP 地址、源端口号、内网 IP 地址26；
