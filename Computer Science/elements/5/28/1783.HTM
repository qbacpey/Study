Title: .symtab

This section holds a symbol table, as "Symbol Table'' in this chapter describes. If a file has a loadable segment that includes the symbol table, the section's attributes will include the SHF_ALLOC bit; otherwise, that bit will be off.

.symtab 中保存着符号表

An object file's symbol table holds information needed to locate and relocate a program's symbolic definitions and references. 

A symbol table index is a subscript into this array. Index 0 both designates the first entry in the table and serves as the undefined symbol index. The contents of the initial entry are specified later in this section
