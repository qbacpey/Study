IPv6 数据报中只有 IPv6 Header 这种头部吗？不一定，有可能一个数据报中可能会有多个 header

IPv6 数据报是通过什么样的方式代替原来 IPv4 Option 字段的功能的？级联头部

IPv6 数据报中级联头部指的是什么？尽管一个数据报中只能有一个固定大小的 IPv6 头部，但是可以有很多拓展头部

IPv6 数据报中的拓展头部指的是什么？不属于 IPv6 头部的其他元数据

IPv6 数据报中的头部是通过什么方式进行组织的？Next Header 字段

Options：IPv6 中将原来 Option 部分的信息单独抽取出来作为拓展头部（Extension header），这些多出来的信息会和其他高层协议的头部级联起来，跟随在 IPv6 头部的后边（IPv6 必须是第一个）
