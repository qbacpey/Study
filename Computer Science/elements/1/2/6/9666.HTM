下面是一个例子（注意 6,7 的广播地址）：

  1. 使用 xxx.xxx.xxx.xxx/xx 形式给出的地址和子网掩码，前边的必然是经典划分出来的地址，也就是说这个地址必然是属于 Class A/B/C 中任意一种的网络号；
  2. 因此，给出的子网掩码相对于 Class 多出来的比特数目就是 “Last Non-Zero Octor” 的数值；
  3. 此时计算 “Magic Number” 就直接使用  减去 “Last Non-Zero Octor” 即可。此后想要计算某个网络的网络号直接使用经典地址加上幻数就能得出，关键在于第一个子网地址和最后一个子网地址要被弃用；
  4. 计算某个网络的广播地址使用下一个子网的网络地址减一即可；
