用户进程只能访问自己的虚拟内存空间

An attempt to access kernel virtual memory causes a page fault, handled by page_fault() in userprog/exception.c, and the process will be terminated.

用户进程的任何针对内核虚拟内存空间的访问都会引发 page fault，page fault 会被 page_fault() 函数所处理（page_fault() 的实现位于 userprog/exception.c 中）用户进程如果引发 pagefault 进程会被直接终止
