Title: Program Loading

程序装载（Program Loading）由谁来完成？操作系统

Program loading is the process by which the operating system creates or augments a process image.

程序装载（Program Loading）指的是什么？操作系统提升或创建进程映像

 The manner in which this process is accomplished and how the page management functions for the process are handled are dictated by the operating system and processor.

程序装载的具体完成方式以及内存页的相关管理功能为操作系统以及处理器所决定
