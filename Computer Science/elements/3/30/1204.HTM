Title: 2.6 Implementing Secure System Calls : stub

We bridge these two views — the user program calling the system call, and the kernel implementing the system call — with a pair of stubs. A pair of stubs is a pair of procedures that mediate between two environments, in this case between the user program and the kernel

用户进程执行系统调用，内核实现系统调用并为用户进程所调用的这整个过程，是通过一对桩（a pair of stub）实现的

系统调用中的一对桩，指的是一对连接用户进程和内核的程序
